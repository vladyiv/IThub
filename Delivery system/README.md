# Система автоматизации службы доставки «Быстрая доставка»

Информационная система для управления заказами и клиентами службы доставки. 

---

##  Ключевые возможности
* **Симметричный GUI-интерфейс**
* **Интерактивный календарь**
* **Полноценный CRUD**
* **Умные фильтры и сортировка**
* **Аналитические отчёты**
* **Импорт/Экспорт данных**
* **Консольный режим (CLI)**
---

## Требования и зависимости
Для работы программы необходим Python версии **3.10+**, а также библиотека для работы с календарём:
```bash
pip install tkcalendar
```
---

## Инструкция по запуску

### 1. Графический режим (GUI)
Чтобы запустить полноценное графическое окно со всеми таблицами и кнопками, запустите скрипт без дополнительных аргументов:
```bash
python database.py
```

### 2. Консольный режим (CLI)
Программа автоматически переключается в скрытый текстовый режим, если обнаруживает аргументы командной строки.
* **Вызов встроенной справки**:
  ```bash
  python database.py --help
  ```
* **Добавление нового клиента через терминал**:
  ```bash
  python database.py --action add_client --name "Никита Студент" --phone "89991112233" --address "Общежитие 2"
  ```
* **Быстрое удаление заказа по ID**:
  ```bash
  python database.py --action delete_order --order_id 3
  ```
* **Экспорт базы в JSON одной командой**:
  ```bash
  python database.py --action export_json
  ```
---

## Структура файлов
* `database.py` — основной исполняемый файл приложения.
* `delivery_tables.db` — файл локальной базы данных SQLite.
* `import.json` — шаблон файла для тестирования функции импорта.
* `README.md` — документация к проекту.
* `Отчёт.txt` и `export.json` появятся автоматически при исполнении связанных с ними функций.
